# Ativos

Este diretório armazena os ativos estáticos aprovados para uso na documentação do SILIC 2.0.

## Exemplos

| Tipo                 | Descrição                                              |
| -------------------- | ------------------------------------------------------ |
| Logotipos            | Identidade visual institucional e variações aprovadas. |
| Ícones               | Elementos gráficos de apoio usados na documentação.    |
| Referências de marca | Materiais de orientação para aplicação da marca.       |
| Visuais exportados   | Imagens e artefatos gráficos prontos para publicação.  |
